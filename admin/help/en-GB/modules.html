Modules Reference

This page summarises JEM modules and their main XML options. It is intended as a local reference and as starter material for the online modules manual.

Event Listing Modules

ModulePurposeMain options
JEM Basic Module Compact list of events. Event set: upcoming, unfinished or archived. Count, no-events message, featured highlighting, hour offset, display order, title length, title visibility, detail links by date/title, venue display/linking, country icon, category and venue filters, date and time formats.
JEM Wide Module Wider event list layout. Event set: upcoming, unfinished, archived or current. Count, no-events message, featured highlighting, hour offset, title length, event/category/venue links, modal use, category and venue filters, date method and date/time formats.
JEM Teaser Module Visual teaser layout with images and descriptions. Event set includes upcoming, unfinished, archived, current and featured. Count, no-events message, hour offset, title length, description display and length, read-more handling, event and venue images, date method, links, modal use, color mode, category and venue colors, category/venue/event/country filters and state matching.
JEM Banner Module Banner/card-style event display. Event set includes upcoming, unfinished, archived, current and open-date events. Count, max days, shuffle and shuffle count, title length, event link, calendar display, color mode, flyer display and flyer link type, description/read-more options, date method, category and venue display/linking, featured-only, category/venue/event filters and state matching.
JEM Jubilee Module Shows past events occurring on the same date. Uses anniversary-style event selection with display options similar to the teaser/banner modules: count, title, descriptions, images, date/time, category and venue display, filters and state matching.

Calendar And Map Modules

ModulePurposeMain options
JEM Calendar Module Small event calendar module. Tooltips and tooltip titles, tooltip event limit, title length, event links, time/category/venue display, day-name length, first weekday, year/month length, month and time offsets, remember navigation, Ajax navigation, current and archived events, category and venue filters, direct link to details and fixed Itemid.
JEM Map Module Displays event or venue markers on a map module position. Auto centre, centre latitude/longitude, zoom, my location, date filter and default date range, full screen map, heat layer, marker files, category and country filters, default country and height. Height default: 500px.

Type Module

ModulePurposeMain options
JEM Types Module Displays JEM event or venue types. Display mode: summary or top N. Top N count. Hide empty types. Show module when there are no matching types.

Advanced Module Options

All modules also inherit standard Joomla module options such as layout, module class suffix, caching and cache time where the module XML exposes them.
