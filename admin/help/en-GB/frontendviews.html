Frontend Views Reference

This page summarises the JEM frontend menu views and the most important options exposed by their menu XML files. It is intended as a local reference and as starter material for the online frontend manual.

Event Views

ViewPurposeMain options and defaults
Events list Lists events with category, venue, date and table controls. Show archived events: No. Featured only: No. Show intro text for events: No. Show icons in order/title/data: Yes. Default ordering: Date, ascending. Date filters support from/until limits. Category filter can include or exclude category IDs and can include subcategories. Open dates can be hidden, shown too, or shown only.
Event detail Displays one selected event. Requires an event selection. Event output then follows the event record, global JEM settings and event-specific display overrides.
Day Shows events for one day or a day offset. Day offset field selects the target day. Featured only: No. Intro text can be shown or hidden. Category switch can include or exclude category IDs. Footer text can be shown or hidden.
Calendar Month calendar view. Intro text can be shown or hidden. Archived events: No. Top category can restrict the tree. Legend positions include no legend, above, below and venue variants. First weekday: Sunday or Monday. Day limit controls visible event items per day. Add, edit and copy event icons can be enabled. Calendar colors include link, background, current day, more-link and featured border colors.
Week calendar Calendar grouped by weeks. Shares calendar options for intro text, archived events, top category, legend, first weekday, day limit, action icons and colors. Number of weeks controls how many weeks are displayed.
Type events Lists events assigned to a selected Event type. Requires a type selection. Show event intro text: No. Show month row: Yes.

Category Views

ViewPurposeMain options and defaults
Categories Lists categories below a selected root category. Root category selection is optional. Show empty categories: Yes. Category count and detail count control how many categories and events are shown. Use category description: Yes. Show empty child categories: No.
Category Shows one category and its events. Requires a category selection. Intro text can be shown or hidden. Use category description: Yes. Show empty child categories: No. Include child events: Yes.

Venue Views

ViewPurposeMain options and defaults
Venue Displays one selected venue and its events. Requires a venue selection. Intro text can be shown or hidden.
Venues Displays grouped venues with event lists. Number of venues controls how many venue groups are shown.
Venues list Lists venues in a table/list layout. Intro text can be shown or hidden. Show state: Yes. Show venue calendar link: Yes. Display order can be venue/city/country, venue/country/city, city/venue/country, city/country/venue, country/venue/city or country/city/venue.
Type venues Lists venues assigned to a selected Venue type. Requires a type selection. Show state: Yes. Show venue calendar link: Yes. Uses the same display-order options as Venues list.

Map Views

ViewPurposeMain options and defaults
Events map Displays event locations on a map. Intro text can be shown or hidden. Auto centre: Yes. Zoom default: 13. Map type: political or physical. My location, full screen and heat layer can be enabled. Date filter is shown by default and can default to all, today, tomorrow, week, month, year or date. Category and country filters can be shown. Height default: 500px.
Venues map Displays venue locations on a map. Intro text can be shown or hidden. Show state: Yes. Auto centre: Yes. Zoom default: 13. Map type: political or physical. My location, full screen, heat layer, category filter and country filter can be enabled. Height can be configured.

User And Search Views

ViewPurposeMain options and defaults
Search Searches JEM events and venues. Intro text can be shown or hidden. Top category can restrict results. Date filter type can use event start date or all event dates. Template suffix can be set. Filter fields can show event type, category, venue, continent, country and dates.
My events Shows events created or editable by the current user. Attendee column: No. Email address display: No.
My venues Shows venues created or editable by the current user. Uses the frontend venue list layout for the current user context.
My attendances Shows registrations for the current user. Registration filter: Yes. Past-registration period controls how far back registrations remain visible.
