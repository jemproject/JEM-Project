List Types

Description

Manage reusable type labels from this screen. Types can be assigned to events, categories or venues and can provide a badge color, an icon and translated display text.

Navigation

To access this screen use: Components → JEM → Types

Toolbar

Select Type(s) by clicking the checkbox(es) at the beginning of each row.

New: Creates a new type.

Edit: Opens the selected type for editing. You can also click the type name.

Publish / Unpublish: Changes whether the type can be used and displayed.

Check In: Unlocks a type that was left checked out by an editor.

Delete: Removes the selected type records.

Columns

Name: The display name of the type. If a description exists, it is shown below the name.

Applies To: Shows whether the type is available for Events, Categories or Venues.

Icon: Displays the Font Awesome icon configured for the type.

Color: Displays the badge color configured for the type.

Access: Joomla access level required to use or see the type where access checks are applied.

Status: Published or unpublished state.

ID: Database identifier of the type.

Filters

Search: Filters the list by type name or description.

Status: Shows all, published or unpublished types.

Applies To: Limits the list to Event, Category or Venue types.
