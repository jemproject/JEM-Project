Online manual: Backend Frontend Modules Plugins

Edit Type

Description

Add or edit a type. Types are optional labels that can be assigned to events, categories and venues.

Navigation

To access this screen use: Components → JEM → Types → New or click an existing type name.

Toolbar

Save: Saves the type.

Save & Close: Saves the type and returns to the type list.

Save & New: Saves the type and opens a new blank type form.

Save as Copy: Saves the current type as a new record.

Cancel / Close: Leaves the edit screen without saving additional changes.

Fields

Name: Required. The display name of the type.

Description: Optional description. The first sentence is used as the tooltip on type badges; the full text can be used as intro text in type views.

Translations: If site languages are installed, the main name and description are used as the base language. Add translated names and descriptions only for languages that need them.

Icon CSS class: Optional Font Awesome 6 class, for example fa-solid fa-calendar. Use the icon category selector and picker or type a class manually.

Color: Optional badge background color. Default: none.

Applies To: Required. Select whether the type is for Events, Categories or Venues. Default: Event.

Status: Published or Unpublished. Default: Published.

Access: Joomla access level. Default: Public.

Language: Content language for the type. Default: All.

Alias: URL-safe alias. Leave blank to generate it automatically from the name.
