Online manual: Backend Frontend Modules Plugins

Plugins Reference

This page summarises the JEM plugins included with the package and the main options exposed by their XML files. It is intended as a local reference and as starter material for the online plugins manual.

Content Plugins

PluginPurposeMain options
Content - JEM Provides JEM content integration helpers. No component-specific configuration fields are exposed in the current XML.
Content - JEM Embed Embeds JEM output in content through plugin syntax or controlled embed calls. Require token: Yes by default. API tokens stores the accepted tokens when token protection is enabled.
Content - JEM List Events Lists JEM events inside articles or content areas. Event set: today, unfinished, upcoming, ongoing, archived, newest, open or all. Featured filter: No, Yes or Only. Title, date, category and venue can be hidden, shown or linked. Title length, date/time formats, end date/time display, category IDs, venue IDs, maximum events and no-events message are configurable.

JEM System Plugins

PluginPurposeMain options
JEM Mailer Sends notification emails for registrations, waiting-list changes, unregistrations, event changes and venue changes. Registration, waiting-list and unregistration notifications can be sent to users, event creators, admins, category recipients and JEM groups. Event notifications cover new events and edited events, including registered users and category ACL recipients. Venue notifications cover new and edited venues. Admin options control automatic admin receiver lookup, explicit admin receiver addresses and HTML email.
JEM Comments Connects JEM with a comments system where available. Comment system: None or JComments.

Search And Dashboard Plugins

PluginPurposeMain options
Finder - JEM Indexes JEM content for Joomla Smart Search. No component-specific configuration fields are exposed in the current XML.
Quick Icon - JEM Adds JEM quick access to the Joomla dashboard. No component-specific configuration fields are exposed in the current XML.

Optional / Third-party Integrations

IntegrationPurposeMain options
AcyMailing Tag JEM Provides JEM event tags for AcyMailing newsletters. Hide past events, show featured only, image display mode, resized image width/height and fixed Itemid.
Community Builder integrations Displays JEM events and attendances in Community Builder contexts. Configuration depends on the installed Community Builder plugin and profile layout.
