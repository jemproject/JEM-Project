Edit Attendee

Description

Manage the registered users from this screen. Here you can select another user as an Attendee to the Event you are currently editing.

Select another user by clicking the Select User text to the right of the User field.

Navigation

To access this screen use: Components → JEM → Events → User count number → User

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

Save: Saves the currently selected user without closing this Edit screen.

Save & Close: Saves the currently selected user, closes this Edit screen and returns to the Registered Users screen.

Cancel: Cancels this operation, closes this Edit screen and returns to the Registered Users screen.

Help: Opens this Help screen.
