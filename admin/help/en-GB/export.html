Export Data

Description

Here you can export data from JEM tables in CSV (Comma Separated Values) format.

Navigation

To access this screen use: Components → JEM → Export Data

Export Options:

Export Events: With this button you can export all Events in a CSV-formatted file (events.csv).

Export Categories: With this button you can export all Categories in a CSV-formatted file (categories.csv).

Export Venues: With this button you can export all Venues in a CSV-formatted file (venues.csv).

Export Cat_Events: With this button you can export all Cat_Events in a CSV-formatted file (cat_events.csv).
