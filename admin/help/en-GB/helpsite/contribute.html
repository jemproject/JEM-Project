Contribute

If you like JEM and you want to help out, you have a lot of options to do so:

Description
Translate Translate JEM in your own language. We use Transifex as a translation platform.
Forum Say thanks, report bugs, suggest new ideas and help others in the forum.
Bugtracker Report bugs and suggest solutions if you have experience with bugtrackers. If you're not that technical, feel free to report bugs in the forum.
Contribute You have skills in PHP, HTML, CSS and MySQL and you wish to contribute? Check the website how you can help
Donate If you just like JEM and want to help us keeping the community alive, consider a donation. However small, we're always happy!.
