CSS Manager

Description

Here you can modify the CSS files of the various views of JEM directly in the backend. Clinking a link on the right will take you to the edit screen for the chosen CSS file.

Attention

Don't change anything if you don't know exactly what you do. Read the comments carefully and backup this file before you alter it.

Navigation

To access this screen use: Components → JEM → Edit CSS

Toolbar

Close: Close the CSS-Manager and go back to the JEM Control Panel.
